## ?? Key Concepts (Simplified)

```
STORED PROCEDURE
?? What: SQL code saved in Oracle database
?? Why: Execute in DB (faster)
?? Example: TAX_GEN_PKG.FinaliseGeneration

REPOSITORY PATTERN
?? What: Class that handles database operations
?? Why: Separates data access from business logic
?? Example: MembershipDataRepository, AdjustmentRequestRepository

PLUGIN / ADDON
?? What: Code loaded dynamically at runtime
?? Why: Can add jobs without recompiling service
?? Example: TaxGenPst loaded as MEF plugin

WINDOWS SERVICE
?? What: Background program running 24/7
?? Why: No user login needed, automatic startup
?? Example: BatchServiceHost.exe

ENUM (Enumeration)
?? What: List of allowed constant values
?? Why: Type-safe mode selection
?? Example: TaxGenerationMode.Annual, .AdjustmentForATO, etc.
```
